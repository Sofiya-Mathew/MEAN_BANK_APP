# MEAN_BANK_APP
### Technologies
* Angular
* Node
* Express
* MongoDB
* JWT
### Features
* User can signin and signup.
* User can deposit and withraw amount from their accounts.
* User can see their transaction history.
* Modern UI design.
### Setup
 #### Server
 * npm install
 * npm start
 #### Client
 * npm install
 * ng serve
